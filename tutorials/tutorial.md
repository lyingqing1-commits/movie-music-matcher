### 🎯 这个工具能做什么？

上传一段**电影/视频素材**和一首**音乐**，AI 会自动：

1. 🔍 **识别电影内容**：AI 利用训练知识识别电影名称、情节、主题
2. 🎨 **分析画面风格**：结合电影知识进行一致性分析（同一电影多次上传结果稳定）
3. 🎵 **分析音乐结构**：识别 intro/verse/chorus/drop 等段落结构
4. 🤖 **智能匹配评分**：音乐与视频风格匹配并打分
5. ✂️ **智能剪辑生成草稿**：自动生成剪映草稿

### 🚀 使用步骤

| 步骤 | 操作 |
|------|------|
| **1. 上传视频** | 点击左侧"电影素材"框，选择视频文件（MP4/MOV 等，最大 2GB） |
| **2. 上传音乐** | 点击右侧"音乐文件"框，选择 1-3 首音乐（MP3/WAV 等） |
| **3. 选择模式** | 在上传区下方选择剪辑模式（视频优先 / 音乐优先） |
| **4. 开始分析** | 点击"开始分析匹配"按钮，AI 自动完成所有步骤 |
| **5. 打开剪映** | 草稿自动生成并复制到剪映目录，在剪映中打开即可 |

### ✂️ 剪辑模式说明

**🎬 视频优先模式**：
- 侧重于视频的"爽点""爆点"
- AI 自动检测场景，评分每个片段的高光度
- 将最精彩的片段放在音乐最重要的段落（Drop/副歌）
- 适合：动作片混剪、高光集锦、燃向剪辑

**🎵 音乐优先模式**：
- 视频剪辑服务于音乐结构
- 平缓片段落在 intro/verse，激烈片段落在 chorus/drop
- AI 根据情绪匹配：action→Drop, calm→Verse, emotional→Bridge
- 适合：MV 风格、文艺混剪、情绪短片

### ✂️ 智能剪辑原理

不同于简单的"节拍等分切割"，v2.0 使用智能剪辑引擎：

1. **电影识别**：AI 识别影片名称，提取情节/主题/名场面知识
2. **场景检测**：FFmpeg 自动检测视频中的自然转场点
3. **高光评分**：AI 对每个场景片段评 0-100 分的高光度
4. **音乐结构**：librosa + AI 分析歌曲的 intro/verse/chorus/drop
5. **智能匹配**：将最佳视频片段精确放置在音乐最合适的段落

### ⚙️ 前置要求

- ✅ **FFmpeg**：程序自动检测，支持系统安装版和剪映自带版
- ✅ **API Key**：已在 config.py 中配置 DeepSeek API
- ✅ **剪映专业版**：用于打开生成的草稿文件
- ✅ **Python 依赖**：librosa（音频分析）、pyJianYingDraft（草稿生成）

### 💡 小贴士

- 同一部电影的不同片段上传，分析结果将保持稳定一致
- 视频优先模式下，高潮片段会自动匹配到音乐的 Drop 段落
- 音乐优先模式下，画面情绪与音乐段落精确对应
- 草稿保存在 `output/` 目录，同时自动复制到剪映草稿目录
- 所有智能功能失败时会自动回退到节拍等分模式，不影响使用
- 上传多首音乐会自动排序，选出最佳匹配
