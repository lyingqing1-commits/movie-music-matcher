📖 使用教程

🎯 这个工具能做什么？

上传一段电影/视频素材和一首音乐，AI 会自动：

  1. 分析视频的画面风格（类型、情绪、色调、节奏）
  2. 分析音乐的音频特征（BPM、能量、调性）
  3. 匹配音乐与视频风格并打分
  4. 按音乐节拍将视频自动切分为多段，生成剪映草稿

🚀 使用步骤

1
上传视频

点击左侧"电影素材"框，选择你的视频文件（MP4/MOV 等，最大 500MB）

2
上传音乐

点击右侧"音乐文件"框，选择 1-3 首音乐（MP3/WAV 等）

3
开始分析

点击"开始分析匹配"按钮，AI 会自动完成所有步骤

4
查看结果 & 打开剪映

分析完成后会显示匹配评分和剪辑建议。草稿文件自动生成，在剪映中打开即可看到视频已被切分为多段

✂️ 视频切分规则

生成草稿时，视频会根据音乐 BPM（节拍）自动切分：

  • 每 2 拍 切一刀（BPM 120 = 每 1 秒一段）
  • 音乐比视频长 → 视频自动循环使用
  • 段与段之间有微小间隙，模拟手动剪辑效果
  • 在剪映中等效于按 Ctrl+B 切分视频

⚙️ 前置要求

  • ✅ FFmpeg：程序会自动检测，无需手动配置
  • ✅ API Key：已在 config.py 中配置 DeepSeek API
  • ✅ 剪映专业版：用于打开生成的草稿文件

💡 小贴士

  • 匹配评分 ≥ 70 分时，草稿会自动添加淡入淡出效果
  • 上传多首音乐会自动排序，选出最佳匹配
  • 草稿保存在 output/ 目录下
  • 如果剪映已安装，草稿会自动复制到剪映草稿目录
